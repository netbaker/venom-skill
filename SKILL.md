# 毒液 - 知识猎食者

## 概述

**毒液**是一个打破信息茧房的知识探索技能。它像毒液的5条触手一样，从多个知识源随机抽取内容，帮助你发现意想不到的知识关联。

**一句话**：「我也不知道想看什么，给我找点有意思的。」

## 触发条件

当用户说以下任意内容时触发：
- "毒液"
- "给我找点有意思的"
- "我不知道想看什么"
- "随机知识"
- "打破信息茧房"
- "知识探险"
- "venom"

## 工作原理

毒液从5个知识源中随机猎食：

| 触手 | 知识源 | 扫描目标 |
|------|--------|----------|
| 🦑 触手1 | 工作区脚本 | `.py`, `.js`, `.sh`, `.bat`, `.html` 文件 |
| 🦑 触手2 | 最近笔记 | `.md` 文件（按修改时间排序） |
| 🦑 触手3 | 个人兴趣 | 用户配置的兴趣关键词 |
| 🦑 触手4 | 随机知识 | 外部知识API或本地知识库 |
| 🦑 触手5 | Dashboard MOC | 索引文件、目录结构、配置文件 |

def calculate_hash(file_path):
    """计算文件的MD5哈希值"""
    hash_md5 = hashlib.md5()
    with open(file_path, "rb") as f:
        for chunk in iter(lambda: f.read(4096), b""):
            hash_md5.update(chunk)
    return hash_md5.hexdigest()

## 高级功能

### 1. 猎食历史
- **存储位置**：`.workbuddy/skills/venom/history.json`
- **记录内容**：猎食时间、触手来源、猎物路径、用户反馈
- **历史限制**：最多保存100条记录
- **用途**：避免重复猎食，分析猎食偏好

### 2. 知识关联
- **关联算法**：基于文件名、内容关键词、目录结构
- **关联类型**：
  - 同目录文件
  - 同类型文件
  - 相似内容文件
  - 历史猎物关联
- **输出**：关联文件列表和关联理由

### 3. 猎食统计
- **统计维度**：
  - 各触手猎食次数
  - 各触手成功率
  - 用户满意度（基于反馈）
  - 猎食时间分布
- **输出格式**：
  ```
  📊 猎食统计
  
  总猎食次数：42
  成功率：95%
  平均满意度：0.78
  
  触手分布：
  - 脚本：15次（36%）
  - 笔记：12次（29%）
  - 兴趣：8次（19%）
  - 知识：5次（12%）
  - MOC：2次（4%）
  ```

### 4. 定时猎食
- **触发方式**：cron 任务或定时器
- **执行时间**：每天早上9点（可配置）
- **执行内容**：随机猎食一次，发送结果到用户
- **配置方式**：在 `interests.json` 中设置

### 5. 进化机制（达尔文进化版）
- **基因系统**：每个触手有权重基因
- **自然选择**：基于用户反馈调整权重
- **基因变异**：10%概率随机调整权重
- **适者生存**：高权重触手更容易被选中

### 6. 多源混合猎食
- **猎食方式**：同时从多个触手猎食
- **展示方式**：组合展示多个猎物
- **关联分析**：分析猎物之间的关联

### 7. AI 解读
- **解读方式**：用 AI 分析猎物内容
- **解读内容**：
  - 代码功能解释
  - 笔记内容总结
  - 知识关联分析
  - 学习建议

### 8. 社交分享
- **分享方式**：导出猎食结果为 Markdown
- **分享内容**：猎物预览、发现提示、关联分析
- **分享平台**：微信、QQ、邮件

### 9. 学习路径

  - 相关文件列表
  - 学习顺序

## 实现方式

毒液技能通过以下步骤实现：

### 1. 扫描工作区
- **工具**：`Glob` 和 `Grep` 工具
- **参数**：
  - 模式：`**/*.py`, `**/*.js`, `**/*.md` 等
  - 排除：`.git`, `node_modules`, `__pycache__`, `.workbuddy`
  - 排序：按修改时间降序（最近优先）
- **输出**：文件路径列表

### 2. 随机选择
- **工具**：Python `random` 模块
- **算法**：
  - 触手选择：基于权重的随机选择（权重越高，选中概率越大）
  - 文件选择：均匀随机选择
- **参数**：
  - 触手权重：可配置，默认相等
  - 选择范围：可配置，默认全部文件

### 3. 内容提取
- **工具**：`Read` 工具
- **参数**：
  - 代码文件：前15行
  - 笔记文件：前8行
  - 配置文件：前10行
  - 编码：UTF-8（自动检测其他编码）
- **输出**：文件内容片段

### 4. 智能摘要
- **工具**：文本处理
- **算法**：
  - 提取文件标题（第一行）
  - 提取关键函数/类名
  - 提取文件描述（docstring）
- **输出**：内容摘要

### 5. 知识关联
- **工具**：文本分析
- **算法**：
  - 关键词提取
  - 相似文件推荐
  - 知识图谱构建
- **输出**：关联提示

## 异常处理

### 常见异常及处理方式

| 异常场景 | 触发条件 | 处理方式 | 用户提示 |
|---------|---------|---------|---------|
| 文件不存在 | 文件被删除或移动 | 跳过该文件，继续猎食下一个 | "文件已移动或删除，跳过..." |
| 权限不足 | 文件被锁定或无读取权限 | 显示提示，跳过该文件 | "无法读取该文件，跳过..." |
| 配置缺失 | interests.json 不存在 | 使用默认兴趣列表 | "使用默认兴趣配置..." |
| 目录为空 | 没有找到任何文件 | 切换到其他触手 | "该目录为空，切换触手..." |
| 编码错误 | 文件编码不是UTF-8 | 尝试其他编码，失败则跳过 | "文件编码异常，跳过..." |
| 网络错误 | 外部知识API不可用 | 使用本地知识库 | "网络不可用，使用本地知识..." |
| 内存不足 | 文件过大无法读取 | 只读取前N行 | "文件较大，只显示部分内容..." |
| 路径过长 | 文件路径超过系统限制 | 跳过该文件 | "路径过长，跳过..." |

### Fallback 路径

当某个触手猎食失败时，按以下顺序尝试：
1. **脚本触手失败** → 尝试笔记触手
2. **笔记触手失败** → 尝试兴趣触手
3. **兴趣触手失败** → 尝试随机知识
4. **随机知识失败** → 尝试MOC触手
5. **所有触手失败** → 显示提示："毒液今天没找到有趣的猎物，明天再来试试！"

**智能降级策略**：
- 如果连续3次猎食同一触手失败，自动降低该触手权重
- 如果所有触手都失败，建议用户检查工作区是否为空
- 如果配置文件损坏，自动重建默认配置

### 错误恢复

- 猎食过程中遇到错误，自动重试最多3次
- 重试失败后，切换到其他触手
- 记录错误日志，便于排查问题
- 错误日志位置：`.workbuddy/skills/venom/error.log`

### 错误日志格式

```
[2026-06-14 23:53:07] ERROR: 触手=脚本, 文件=xxx.py, 错误=文件不存在
[2026-06-14 23:53:08] WARN: 触手=脚本, 连续失败=2, 自动切换到笔记触手
[2026-06-14 23:53:09] INFO: 触手=笔记, 猎物=yyy.md, 状态=成功
```
